Frequentes

Winux OS é um sistema operacional Linux completo baseado em Ubuntu 24.04 LTS. Ele combina uma interface elegante estilo Windows 11 com o poder do Linux, otimizado para gaming (Wine, Proton, DXVK) e desenvolvimento de software (Java, Node.js, Python, Docker e mais).

Requisitos mínimos: CPU x86_64 com SSE4.2, 4GB de RAM, 30GB de espaço em disco e GPU com suporte Vulkan 1.1. Recomendado: 16GB RAM, SSD NVMe de 100GB, e GPU NVIDIA RTX ou AMD RX 6000+ para melhor experiência em gaming.

Sim! O Winux OS é completamente gratuito e open-source. Você pode baixar a ISO, instalar em quantos computadores quiser, modificar e distribuir livremente. O código-fonte está disponível no GitHub.

Sim! O Winux vem com Wine 9.0, Proton GE, DXVK e VKD3D pré-configurados. Você pode instalar Steam, Epic Games, GOG e jogar a maioria dos games Windows. Lutris também está incluso para facilitar a instalação de jogos.

Baixe a ISO, grave em um pendrive USB usando Rufus (Windows) ou dd (Linux), reinicie o computador pelo USB e siga o instalador gráfico Calamares. O processo completo leva cerca de 15-20 minutos.
