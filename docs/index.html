FAQ

Perguntas
Frequentes

Winux OS é uma solução que permite executar aplicativos Linux nativamente no Windows usando WSL2 (Windows Subsystem for Linux 2). Ele oferece uma interface moderna e intuitiva para gerenciar e usar aplicativos Linux sem precisar de dual-boot ou máquinas virtuais tradicionais.

Para usar o Winux, você precisa de: Windows 10 versão 2004 ou superior (Build 19041+) ou Windows 11, processador de 64 bits, pelo menos 4GB de RAM (8GB recomendado), e aproximadamente 2GB de espaço em disco. A virtualização deve estar habilitada na BIOS.
