FAQ

Perguntas
Frequentes

Winux OS v1.2 Blaze e a versao mais avancada do sistema operacional Linux para gamers e desenvolvedores. Inclui AI Assistant com GPT-4o, sincronizacao na nuvem com 6 provedores, integracao com smartphone via Winux Connect, 48 apps nativos em Rust/GTK4 e muito mais.

Sim, o AI Assistant usa Azure OpenAI. Voce precisa criar uma conta Azure gratuita e obter uma chave de API. Configure em ~/.config/winux/ai.toml ou via variaveis de ambiente WINUX_AI_API_KEY e WINUX_AI_ENDPOINT.

Instale o app Winux Connect no seu Android (disponivel na Play Store ou APK incluso). Abra o app no desktop (Menu > Winux Connect), escaneie o QR code com o telefone e pronto! Os dispositivos se conectam automaticamente via Wi-Fi local.

O Winux Cloud suporta Google Drive, Microsoft OneDrive, Dropbox, Amazon S3, Azure Blob Storage e qualquer servidor WebDAV. Voce pode configurar multiplas contas e sincronizar pastas diferentes com cada provedor.

Minimo: CPU x86_64 com SSE4.2, 4GB RAM, 40GB disco, GPU com Vulkan 1.1. Recomendado: 16GB RAM, SSD NVMe 100GB+, GPU NVIDIA RTX ou AMD RX 6000+ para gaming e aceleracao de IA.
